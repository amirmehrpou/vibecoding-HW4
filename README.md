# 🌍 سلطه جهانی (World Dominion)

یک بازی استراتژیک دو نفره با استفاده از داده‌های واقعی کشورهای جهان

## معرفی

World Dominion یک بازی کارتی دونفره است که در آن هر کشور به یک کارت قابل
بازی تبدیل می‌شود.

اطلاعات کشورها به‌صورت زنده از REST Countries API دریافت می‌شوند و
بازیکنان با استفاده از کشورها، امتیاز کسب می‌کنند، کشورها را تسخیر می‌کنند
و برای سلطه بر جهان رقابت می‌کنند.

هر کارت شامل:

-   نام کشور
-   پرچم
-   پایتخت
-   منطقه
-   جمعیت
-   قدرت
-   توانایی ویژه

------------------------------------------------------------------------

## ویژگی‌ها

### 🌎 داده‌های واقعی

تمام کارت‌ها از API زیر تولید می‌شوند:

https://restcountries.com

### ⚔️ سیستم قدرت

قدرت هر کشور از روی جمعیت محاسبه می‌شود:

Power = floor(log10(population))

مثال:

-   1,000,000 نفر → قدرت 6
-   10,000,000 نفر → قدرت 7
-   100,000,000 نفر → قدرت 8

------------------------------------------------------------------------

### 🌐 پاداش منطقه

اگر بازیکن:

-   ۲ کارت از یک منطقه داشته باشد → +1 قدرت
-   ۳ کارت یا بیشتر از یک منطقه داشته باشد → +2 قدرت

------------------------------------------------------------------------

### 🏛 توانایی پایتخت‌ها

بر اساس حرف اول پایتخت:

  حرف اول   توانایی
  --------- -----------------
  A-F       ⚔️ کشور مهاجم
  G-L       🛡️ کشور مدافع
  M-R       🎴 کشور پشتیبان
  S-Z       ⭐ کشور افتخار

⚔️ کشور مهاجم: +1 هنگام حمله

🛡️ کشور مدافع: +1 هنگام دفاع

🎴 کشور پشتیبان: پس از نبرد ۱ کارت می‌کشد

⭐ کشور افتخار: هنگام برد +1 امتیاز اضافه می‌گیرد

------------------------------------------------------------------------

### 😴 قانون خستگی

بعد از حمله، کارت خسته می‌شود.

کارت خسته:

-   نمی‌تواند حمله کند
-   می‌تواند دفاع کند
-   می‌تواند هدف حمله قرار بگیرد
-   در شروع نوبت بعدی دوباره آماده می‌شود

------------------------------------------------------------------------

## روند بازی

### آماده‌سازی

1.  کشورها از API دریافت می‌شوند.
2.  دسته کارت‌ها بر زده می‌شود.
3.  هر بازیکن ۵ کارت دریافت می‌کند.
4.  بازیکن اول شروع می‌کند.

### هر نوبت

1.  کارت‌های خسته آماده می‌شوند.
2.  یک کارت جدید به صورت خودکار کشیده می‌شود.
3.  مهاجم انتخاب می‌شود.
4.  مدافع انتخاب می‌شود.
5.  نبرد حل می‌شود.
6.  کارت مهاجم خسته می‌شود.
7.  نوبت به بازیکن بعدی می‌رسد.

------------------------------------------------------------------------

### 🌍 تسخیر ۱۰ کشور

یا

### 📦 پایان بازی

وقتی دسته خالی شود و یک بازیکن دیگر کارتی نداشته باشد:

-   بیشترین امتیاز برنده است
-   در صورت تساوی، بیشترین کشور تسخیرشده برنده است
-   در صورت تساوی کامل، بازی مساوی است
